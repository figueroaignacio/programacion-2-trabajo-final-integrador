# Trabajo Final Integrador (TFI) - Aplicación Java con DAO y MySQL

## Descripción
Este proyecto corresponde al **Trabajo Final Integrador (TFI)** de la materia de Programación. La aplicación está desarrollada en **Java 21** y modela dos clases relacionadas mediante una **asociación unidireccional 1→1**, donde la clase `A` referencia a la clase `B`.
